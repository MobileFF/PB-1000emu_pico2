# 利用ガイド

このガイドでは、PB-1000 エミュレータのセットアップ方法、起動方法、および操作方法について説明します。

## 1. 初期設定

### ROM イメージ
エミュレータが動作するためには、オリジナルのカシオ PB-1000 ROM イメージが必要です。これらはプロジェクトに含まれていません。
- `rom0.bin`: 内部 ROM (6KB、アドレス 0x0000 - 0x17FF)
- `rom1.bin`: 外部 ROM (32KB、アドレス 0x8000 - 0xFFFF)

これらのファイルを Pico の内部フラッシュの `/roms/` ディレクトリ、または SD カードのルートに配置してください。

### ディレクトリ構造 (SD カード/フラッシュ)
```text
/
├── main.py             # エントリポイント
├── roms/               # バックアップ用 ROM/RAM ストレージ
│   ├── rom0.bin
│   └── rom1.bin
└── sd/                 # 推奨ストレージ (SD カード使用時)
    ├── rom0.bin        # /roms/ より優先されます
    ├── rom1.bin
    ├── ram0.bin        # 保存された RAM 状態
    ├── regs.json       # 保存されたレジスタの状態
    └── screenshots/    # キャプチャされた PBM 画像
```

## 2. エミュレータの起動

1.  ILI9341 ディスプレイと USB キーボード (OTG 経由) を接続します。
2.  Pico 2 の電源を入れます。
3.  `main.py` が正しくアップロードされていれば、ディスプレイに PB-1000 の「ベゼル」が表示され、エミュレータが起動します。
4.  システムは以前の状態 (`ram0.bin`, `regs.json`) を復元しようとします。見つからない場合は、コールドブート（新規起動）を行います。

## 3. キーボード操作

### 特殊機能キー
| PC キー | PB-1000 の機能 | 備考 |
| :--- | :--- | :--- |
| **F9** | **Reset** | ハードウェアリセット (PC=0x0001) |
| **F11** | **Save State** | 手動での RAM およびレジスタの状態保存 |
| **PrintScreen** | **Screenshot** | 現在の LCD 内容を SD に `.pbm` として保存 |
| **Esc / Ctrl+C** | **BREAK** | BASIC プログラムの停止 / エラー解除 |
| **Enter** | **EXE** | コマンドの実行 |
| **Backspace** | **BS** | 文字の削除 |
| **Insert** | **INS** | 挿入モード |
| **矢印キー** | **Cursors** | カーソルの移動 |
| **F1** - **F4** | **T1** - **T4** | ファンクションキー (T1-T4) |
| **Alt** (L/R) | **Shift** | PB-1000 の Shift キー |

### キーマッピングの詳細
エミュレータは、標準的な HID キーボードのスキャンコードを PB-1000 のマトリックスにマッピングします。
- アルファベットと数字: 直接マッピング。
- 記号: PB-1000 の同等品にマッピングされます (例: PC の `Shift + 2` -> PB-1000 の `"` )。
- **MENU**, **CAL**, **LCKEY**: それぞれ `F5`, `F6`, `F7` にマッピングされています（正確な割り当てについては `keymap.py` を参照してください）。

## 4. タッチインターフェース

PB-1000 の有名な 16 キー・タッチパネル (T1-T16) は、LCD のタッチスクリーンを介してエミュレートされます。
- LCD 領域をタップすると、対応するタッチキー (T1-T16) がトリガーされます。
- 192x32 の LCD 領域は、タッチ用に 4x4 のグリッドに分割されています。

## 5. ステート（状態）管理

エミュレータは、**F11** キーを押すと自動的に状態を保存します。
- **RAM**: `ram0.bin` (内部 8KB) および `ram1.bin` (拡張 32KB が有効な場合)。
- **レジスタ**: `regs.json` は、CPU プログラムカウンタ、フラグ、および汎用レジスタを保存します。
- **自動ロード**: 起動時に、エミュレータは SD カード (またはフラッシュ) 上のこれらのファイルを探し、セッションを継続するために復元します。

## 6. スクリーンショット

**PrintScreen** キーを押すと、現在の LCD 内容がキャプチャされます。
- 形式: PBM (Portable BitMap、1 ビット白黒)。
- 場所: `/sd/screenshots/screenshot_YYYYMMDD_HHMMSS.pbm`。
- デバッグ用に、生の VRAM ダンプ (`vram_dump_....bin`) も保存されます。
